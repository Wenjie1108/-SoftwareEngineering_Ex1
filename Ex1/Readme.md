## How to Submit to the Instructor Repository

⚠️ IMPORTANT RULES
- You MUST fork this repository
- You MUST upload files ONLY inside your own folder
- You MUST NOT modify or delete any other files or folders
- Pull requests that modify other files will be rejected

---

### Step 1: Fork the Repository
1. Click **Fork** (top-right of this page)
2. This creates your own copy of the repository

---

### Step 2: Add Your Folder
1. Open the `Ex1` folder
2. Create a new folder named:
   - `StudentID_FullName`
3. Upload your Visual Studio project files into your folder

---

### Files You Must REMOVE Before Submitting
Do NOT include:
- `bin/`
- `obj/`
- `.vs/`

---

### Step 3: Commit Your Changes
- Commit directly to your fork

---

### Step 4: Open a Pull Request
1. Go to **Pull Requests**
2. Click **New Pull Request**
3. Submit the pull request to the instructor repository
